Сюжет игры:
Игрок попадает в холл, где ему объявляются правила игры, суть - выбраться из замка живым.
Если осмотреться, можно найти 3 пути: Выход из игры, вход на крышу, дверь в следующую комнату. Все остальные выходы закрыты, кроме двери. Рядом с выходом из игры есть планшет, который выключен. Выход на крышу находится высоко на потолке. Рядом находится надпись "Выход на крышу в плохую погоду очень опасен". Если побегать по холлу, можно найти бумажку с цифрой.
Дверь ведет в лабиринт. Внутри лабиринта можно найти:
гаечный ключ (для генератора)
бумажка с цифрой
факел 

Если найти выход из лабиринта, можно попасть на склад, на котором можно найти бомбу, бумажку с цифрой, ключ, который открывает дверь из холла, и лестница. Если воспользоваться лестницей в холле, то можно попасть на чердак, на котором можно найти запал. Запал можно использовать, чтобы зажечь факел и бомбу. 

Открывая дверь из холла с помощью найденного ключа, можно попасть в темный коридор. Но пройти туда можно только при наличии зажжённого факела. В темном коридоре есть бумажка с цифрой и два выхода, один из которых является ловушкой. Если у героя нет с собой бомбы, то он проиграл. Если есть, то он выходит. Второй выход ведет в комнату с генератором, для включения которого требуется гаечный ключ. После включения начинает работать планшет у выхода из игры. Туда надо ввести найденные цифры, после чего герой выходит из игры и побеждает.
